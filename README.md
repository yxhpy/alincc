# alincc

一个聚焦 **Claude Code 代购 / 逆向 / 共享低价站** 的轻量静态站点。

## 当前版本定位
这版只看：

- 代购
- 代充
- 共享账号
- 拼车 / 合租
- 中转 / 逆向味平台

不再混入：
- 官方方案
- 正规 API 平台
- 正常云厂商接入方案

## 当前页面
- `index.html`：Claude Code 代购 / 逆向 / 共享低价站 Top10（2026）
- `styles.css`：站点样式
- `script.js`：轻量交互
- `robots.txt`：抓取规则
- `sitemap.xml`：站点地图

## 页面核心原则
- 低价优先
- 只看野路子平台
- 风险明牌
- 不装成安全白名单
